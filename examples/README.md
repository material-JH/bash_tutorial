# 예제 파일 모음

튜토리얼에서 직접 만들지 않고 복사해서 사용할 수 있는 실습 파일입니다.

원격 서버에서 저장소를 받은 뒤:

```bash
server$ mkdir -p ~/linux-practice
server$ cp -r ~/bash_tutorial/examples ~/linux-practice/
server$ cd ~/linux-practice/examples
```

주요 폴더:

- `navigation/`: `cd`, `ls`, `tree` 연습용 폴더 구조
- `text/`: `cat`, `more`, `vim`, `head`, `tail`, `grep`, `wc` 연습용 텍스트
- `files/`: `cp`, `mv`, `rm` 연습용 파일
- `scripts/`: 실행 권한과 스크립트 실행 연습
- `qe/`: Quantum ESPRESSO 입력/출력/로그/가짜 실행 파일 예제
- `binary/`: 바이너리 파일 확인 연습용 샘플
